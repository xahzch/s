## 技术实现

### CSS 类命名规范
```css
/* 移动端优先 */
.base-class

/* 电脑端适配 */
.desktop:class-name

/* 大屏电脑端 */
.lg-desktop:class-name
```

### 关键断点
- `desktop: 768px` - 平板和小屏电脑
- `lg-desktop: 1024px` - 大屏电脑
- `xl-desktop: 1280px` - 超大屏

### 布局策略
- 移动端：单列垂直布局
- 电脑端：多列网格布局
- 响应式容器：`max-w-[420px] desktop:max-w-[800px] lg-desktop:max-w-[1200px]`
